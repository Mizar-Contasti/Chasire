Templates are used to show amazing UI/UX components build with strong blocks and patterns.

Please check first Blocks and let's go.

```shell
Chasire
└─── templates
    │   webTemplates
    │   fbTemplates
    │   wsTemplates
    │   tgTemplates
    │   templates
```
